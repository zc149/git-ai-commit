# Git AI Commit – 프로젝트 개요

## 프로젝트 목적

Git AI Commit은 git diff를 분석하여 자동으로 Conventional Commit 메시지를 생성해주는 CLI 도구입니다.

개발자가 더 이상 커밋 메시지를 고민하지 않아도 되도록 만드는 것이 목표입니다.

---

## 해결하려는 문제

현재 개발자들이 겪는 문제:

- 커밋 메시지를 매번 고민해야 함
- diff를 직접 읽고 요약해야 함
- 팀 컨벤션을 지키기 어려움
- 메시지 품질이 사람마다 다름

---

## 핵심 기능

- staged diff 자동 분석
- Conventional Commit 형식 메시지 자동 생성
- 여러 개의 후보 제안
- 사용자가 선택하면 즉시 git commit 실행

---

## 사용 흐름

1. git add로 파일 스테이징  
2. git ai-commit 실행  
3. 생성된 메시지 중 하나 선택  
4. 자동 커밋 완료  

---

## 대상 사용자

- Git을 사용하는 모든 개발자
- CLI 환경에서 개발하는 사용자
- 커밋 메시지 작성이 귀찮은 개발자
